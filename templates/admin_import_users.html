CSV Format Guide

Minimum required columns:

Column Required? Notes
username Yes Must be unique
name Yes Full name
password Yes Initial password
user_type Yes admin, lecturer, or student
email Optional Recommended (unique)

Role-specific optional columns:

  • Student: student_id, major, enrollment_year
  • Lecturer: employee_id, department, office_location, office_hours
  • Admin: role (optional)
